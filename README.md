# WH40K Lore Archive

**Warhammer 40,000 Grimdark Lore Archive**

> "There is only war."

**[Visit the Live Site →](https://wh40k-lore.vercel.app/)**

A comprehensive, immersive archive of Warhammer 40,000 lore, cosmology, history, and factions. Built with Next.js, MDX, and a full grimdark aesthetic.

## Features
- Deep dives into Cosmology, The Imperium, Primarchs, Chaos, and Xenos
- Interactive timeline
- Global search (Cmd/Ctrl + K)
- Rich MDX articles

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind + shadcn/ui
- Framer Motion
- MDX
